A member can request to borrow an item by submitting: desired time window (start/end dates) and their phone number. Owner receives push notification + email. Both parties must confirm the borrow in-app (dual confirmation). Once confirmed, item availability is automatically set to unavailable for that window. Owner can override availability at any time. The agreed time window is recorded in case of disputes but items do not auto-return - someone must confirm return. Multiple people can request the same item simultaneously.
